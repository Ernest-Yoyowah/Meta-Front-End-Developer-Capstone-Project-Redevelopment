# Little Lemon Restaurant Online Reservation System

## Overview
Welcome to the Little Lemon Restaurant App! This is a capstone project for the Meta Front End Development course, developed using modern web technologies to enhance the online reservation experience.

## Highlights
- Revitalized an existing Capstone Project with a fresh outlook.
- Introduced innovative features, enhanced design, and improved user interactions.
- Utilized cutting-edge technologies to ensure optimal performance.

## Features
- [x] Explore restaurant information, including an enticing overview, special menu items, and customer testimonials.
- [x] Browse available reservation times based on selected dates and party sizes.
- [x] Seamlessly make reservations by selecting preferred time slots and providing essential contact details.
- [x] Embraced a clean and intuitive UI design that fosters a user-friendly experience.
- [x] Implemented responsive design to ensure a seamless experience across various devices.
